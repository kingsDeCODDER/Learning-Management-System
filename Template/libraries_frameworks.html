Python Libraries and Frameworks

What are Libraries and Frameworks?

Libraries are collections of functions and methods that you can use in your programs. Frameworks are more comprehensive collections that provide the structure for building applications.

Popular Python Libraries

Here are some of the most popular libraries in Python:

NumPy

NumPy is used for numerical computing in Python. It provides support for arrays, matrices, and many mathematical functions. 

import numpy as np
arr = np.array([1, 2, 3, 4])
print(arr)

Pandas

Pandas is used for data manipulation and analysis. It provides data structures like DataFrames for handling tabular data. 

import pandas as pd
data = {'Name': ['John', 'Anna'], 'Age': [28, 24]}
df = pd.DataFrame(data)
print(df)

Matplotlib

Matplotlib is a plotting library used for creating static, animated, and interactive visualizations in Python. 

import matplotlib.pyplot as plt
plt.plot([1, 2, 3], [4, 5, 6])
plt.show()

Popular Python Frameworks

Python frameworks provide the foundation for building applications, particularly web applications.

Django

Django is a high-level web framework that encourages rapid development. It follows the "Don't Repeat Yourself" (DRY) principle. 

# In Django, you define models and views
from django.db import models
class Post(models.Model):
    title = models.CharField(max_length=100)
    content = models.TextField()

Flask

Flask is a micro web framework that is lightweight and flexible, giving you more control over your application structure. 

from flask import Flask
app = Flask(__name__)

@app.route('/')
def home():
    return "Hello, Flask!"

FastAPI

FastAPI is a modern, fast (high-performance) web framework for building APIs with Python. 

from fastapi import FastAPI
app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Hello, FastAPI!"}
