2. Control Flow

Question: Write a program to check if a number is even or odd.

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter a number: ");
        int num = scanner.nextInt();

        if (num % 2 == 0) {
            System.out.println(num + " is even.");
        } else {
            System.out.println(num + " is odd.");
        }
    }
}

Practice: Modify the program to check if a number is prime.

3. Arrays

Question: Write a Java program to find the largest number in an array.

public class Main {
    public static void main(String[] args) {
        int[] numbers = {5, 3, 9, 7, 1};
        int largest = numbers[0];

        for (int i = 1; i < numbers.length; i++) {
            if (numbers[i] > largest) {
                largest = numbers[i];
            }
        }

        System.out.println("The largest number is: " + largest);
    }
}

Practice: Modify the program to find the smallest number in the array.

4. Object-Oriented Programming (OOP)

Question: Write a program with a Car class that has fields for make, model, and year. Create a method displayInfo() to display the car's information.

class Car {
    String make;
    String model;
    int year;

    Car(String make, String model, int year) {
        this.make = make;
        this.model = model;
        this.year = year;
    }

    void displayInfo() {
        System.out.println("Make: " + make);
        System.out.println("Model: " + model);
        System.out.println("Year: " + year);
    }
}

public class Main {
    public static void main(String[] args) {
        Car car = new Car("Toyota", "Corolla", 2020);
        car.displayInfo();
    }
}

Practice: Add another method to the Car class that calculates and returns the car's age based on the current year.

5. Inheritance

Question: Create a class Animal with a method sound(). Then, create a subclass Dog that overrides the sound() method to print "Bark".

class Animal {
    void sound() {
        System.out.println("Some sound...");
    }
}

class Dog extends Animal {
    @Override
    void sound() {
        System.out.println("Bark");
    }
}

public class Main {
    public static void main(String[] args) {
        Dog dog = new Dog();
        dog.sound();  // Outputs: Bark
    }
}

Practice: Create another subclass Cat that overrides the sound() method to print "Meow".

6. Exception Handling

Question: Write a program that handles a division by zero exception using a try-catch block.

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        try {
            System.out.print("Enter dividend: ");
            int dividend = scanner.nextInt();

            System.out.print("Enter divisor: ");
            int divisor = scanner.nextInt();

            int result = dividend / divisor;
            System.out.println("Result: " + result);
        } catch (ArithmeticException e) {
            System.out.println("Cannot divide by zero!");
        }
    }
}

Practice: Add another catch block to handle InputMismatchException when the user enters a non-integer value.

7. Java Collections
