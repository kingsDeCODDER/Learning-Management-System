Python Basic Syntax

Introduction

Python is a versatile, high-level programming language that is widely used for web development, data analysis, artificial intelligence, scientific computing, and more. Its syntax is designed to be readable and straightforward, making it a great choice for beginners.

Comments

Comments are used to explain code and make it more readable. They are ignored by the Python interpreter. 

# This is a single-line comment

For multi-line comments, you can use triple quotes:

"""
This is a
multi-line comment
"""

Variables

Variables are used to store information that can be referenced and manipulated later in the program. In Python, variables are created when you assign a value to them: 

x = 5
y = "Hello, World!"

Python is dynamically typed, so you don't need to declare the type of a variable explicitly.

Indentation

Indentation is crucial in Python. It is used to define the structure and flow of your code, particularly for blocks like loops and conditionals: 

if x > 0:
    print("x is positive")

Make sure you use consistent indentation (typically four spaces) throughout your code.

Data Types

Python supports various data types including:

  • int - Integer numbers
  • float - Floating-point numbers
  • str - Strings (text)
  • bool - Boolean values (True or False)
  • list - Ordered sequences of values
  • dict - Key-value pairs

Example:

age = 30             # int
height = 5.9          # float
name = "Alice"        # str
is_student = True     # bool
colors = ["red", "green", "blue"]  # list
person = {"name": "Alice", "age": 30}  # dict

Control Flow

Control flow in Python is managed using conditional statements like if, elif, and else, and loops like for and while.

Example of an if statement:

if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

Example of a for loop:

for color in colors:
    print(color)

Functions

Functions are reusable blocks of code that perform a specific task. You define a function using the def keyword: 

def greet(name):
    return "Hello, " + name

To call a function, you use its name followed by parentheses:

greeting = greet("Alice")
print(greeting)  # Output: Hello, Alice
