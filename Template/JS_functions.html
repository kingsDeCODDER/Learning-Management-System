JavaScript Functions

Function Declarations and Expressions

Functions in JavaScript allow you to group code that can be reused. You can create functions using declarations or expressions.

Function Declaration

A function declaration defines a named function:

function greet() {
    console.log("Hello, world!");
}

greet(); // Output: Hello, world!

Function Expression

A function expression stores a function in a variable:

const greet = function() {
    console.log("Hello, world!");
};

greet(); // Output: Hello, world!

Arrow Functions (ES6)

Arrow functions, introduced in ES6, provide a more concise syntax for writing functions.

// Traditional function
const add = function(a, b) {
    return a + b;
};

// Arrow function
const addArrow = (a, b) => a + b;

console.log(add(2, 3)); // Output: 5
console.log(addArrow(2, 3)); // Output: 5

Arrow functions have some key differences from traditional functions, particularly in how they handle the this keyword.

Callbacks

A callback function is a function passed into another function as an argument, which is then invoked inside the outer function.

Callbacks are often used in asynchronous operations like handling events or performing actions after a time delay.

// Example of a callback function
function greet(name, callback) {
    console.log("Hello, " + name);
    callback();
}

function sayGoodbye() {
    console.log("Goodbye!");
}

greet("Alice", sayGoodbye);
// Output:
// Hello, Alice
// Goodbye!

Closures

A closure is a function that remembers the variables from its outer scope even after the outer function has completed execution. This allows you to maintain access to the outer function's variables.

// Example of a closure
function outerFunction() {
    let outerVariable = "I'm from outer function";

    return function innerFunction() {
        console.log(outerVariable);
    };
}

const closure = outerFunction();
closure(); // Output: I'm from outer function

In this example, the innerFunction() is able to remember and access the outerVariable even after outerFunction() has finished executing.
