Java Basic Syntax

Introduction

Java syntax is the set of rules that defines the structure of Java programs. Understanding these rules, including how to use data types and operators, is fundamental to writing correct Java code.

Basic Structure of a Java Program

Every Java program consists of a class definition and a main method where execution begins. Here's an example of a simple Java program: 


public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

In this program:

  • public class HelloWorld: Defines a class named HelloWorld.
  • public static void main(String[] args): The main method, which serves as the entry point for Java applications.
  • System.out.println(): Outputs text to the console.

Data Types

Java supports different types of data that you can work with. These data types fall into two categories: primitive and non-primitive.

Primitive Data Types

  • int: Represents integers (e.g., int age = 25;).
  • double: Represents floating-point numbers (e.g., double price = 19.99;).
  • char: Represents single characters (e.g., char grade = 'A';).
  • boolean: Represents true or false values (e.g., boolean isJavaFun = true;).

Non-Primitive Data Types

  • String: Represents sequences of characters (e.g., String name = "John";).
  • Arrays: Used to store multiple values of the same type (e.g., int[] numbers = {1, 2, 3};).

Variables and Constants

Variables are containers for storing data values. In Java, you declare a variable using its data type: 


int age = 30;
String name = "Alice";
boolean isActive = true;

Constants, on the other hand, are variables whose values cannot be changed. You use the final keyword to declare constants: 


final int MAX_USERS = 100;

Operators

Java provides a variety of operators to perform operations on variables and values.

Arithmetic Operators

  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • /: Division
  • %: Modulus (remainder)

Comparison Operators

  • ==: Equal to
  • !=: Not equal to
  • >: Greater than
  • <: Less than
  • >=: Greater than or equal to
  • <=: Less than or equal to

Logical Operators

  • &&: Logical AND
  • ||: Logical OR
  • !: Logical NOT

Comments in Java

Comments are used to make code more readable. Java supports single-line and multi-line comments: 


// This is a single-line comment

/* This is
   a multi-line comment */

Basic Control Structures

Control structures are used to manage the flow of execution in a program.

If-Else Statements


int age = 20;
if (age >= 18) {
    System.out.println("You are an adult.");
} else {
    System.out.println("You are a minor.");
}

For Loops


for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

While Loops


int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}

Conclusion

Java's basic syntax provides the foundation for writing more complex programs. By understanding data types, operators, variables, and control structures, you can begin to develop functional Java applications.
