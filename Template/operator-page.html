Python Operators

Introduction to Operators

Operators are special symbols in Python that carry out arithmetic or logical computation. The value that the operator operates on is called the operand.

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations like addition, subtraction, multiplication, and division. 


Addition:       +       (e.g., 5 + 2 = 7)
Subtraction:    -       (e.g., 5 - 2 = 3)
Multiplication: *       (e.g., 5 * 2 = 10)
Division:       /       (e.g., 5 / 2 = 2.5)
Floor Division: //      (e.g., 5 // 2 = 2)
Modulus:        %       (e.g., 5 % 2 = 1)
Exponent:       **      (e.g., 5 ** 2 = 25)

Comparison Operators

Comparison operators are used to compare two values. They return either True or False based on the comparison. 


Equal:          ==      (e.g., 5 == 5 is True)
Not Equal:      !=      (e.g., 5 != 3 is True)
Greater Than:   >       (e.g., 5 > 2 is True)
Less Than:      <       (e.g., 5 < 2 is False)
Greater or Equal: >=    (e.g., 5 >= 5 is True)
Less or Equal:  <=      (e.g., 5 <= 2 is False)

Logical Operators

Logical operators are used to combine conditional statements.


and:    Returns True if both statements are true (e.g., 5 > 2 and 5 < 10 is True)
or:     Returns True if one of the statements is true (e.g., 5 > 2 or 5 < 3 is True)
not:    Reverses the result, returns False if the result is true (e.g., not(5 > 2) is False)

Assignment Operators

Assignment operators are used to assign values to variables.


Assign:         =       (e.g., x = 5)
Add and Assign: +=      (e.g., x += 3 is equivalent to x = x + 3)
Subtract and Assign: -= (e.g., x -= 3 is equivalent to x = x - 3)
Multiply and Assign: *= (e.g., x *= 3 is equivalent to x = x * 3)
Divide and Assign:   /= (e.g., x /= 3 is equivalent to x = x / 3)

Bitwise Operators

Bitwise operators are used to compare (binary) numbers.


AND:    &       (e.g., 5 & 3)
OR:     |       (e.g., 5 | 3)
XOR:    ^       (e.g., 5 ^ 3)
NOT:    ~       (e.g., ~5)
Left Shift: <<  (e.g., 5 << 1)
Right Shift: >> (e.g., 5 >> 1)
