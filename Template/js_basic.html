JavaScript Basics

Variables and Data Types

In JavaScript, variables are containers for storing data values. You can declare variables using var, let, or const.

var

Used in older code, function-scoped.

var x = 10;
console.log(x); // Output: 10

let

Introduced in ES6, block-scoped.

let y = 20;
console.log(y); // Output: 20

const

Constant variable, cannot be reassigned.

const z = 30;
console.log(z); // Output: 30

JavaScript supports various data types:

  • String: text, e.g., "Hello"
  • Number: numeric values, e.g., 42
  • Boolean: true or false
  • Null: intentional absence of any object value
  • Undefined: variable declared but not assigned a value
  • Object: complex data structures

Operators

Arithmetic Operators

Used for mathematical calculations.

  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • /: Division
  • %: Modulus (Remainder)
  • ++: Increment
  • --: Decrement
let a = 10;
let b = 5;
console.log(a + b); // Output: 15

Comparison Operators

Used to compare values.

  • ==: Equal to
  • !=: Not equal to
  • ===: Strict equal to
  • !==: Strict not equal to
  • >: Greater than
  • <: Less than
  • >=: Greater than or equal to
  • <=: Less than or equal to
console.log(a == b); // Output: false
console.log(a > b); // Output: true

Logical Operators

Used to combine conditional statements.

  • &&: Logical AND
  • ||: Logical OR
  • !: Logical NOT
let c = true;
let d = false;
console.log(c && d); // Output: false
console.log(c || d); // Output: true

Control Flow

if / else Statements

Used to execute code based on a condition.

let age = 18;
if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}

switch Statements

Used to perform different actions based on different conditions.

let fruit = "apple";
switch (fruit) {
    case "banana":
        console.log("Banana is yellow.");
        break;
    case "apple":
        console.log("Apple is red.");
        break;
    default:
        console.log("Unknown fruit.");
}

Loops

Loops allow you to execute a block of code multiple times.

for Loop

for (let i = 0; i < 5; i++) {
    console.log("Iteration " + i);
}

while Loop

let i = 0;
while (i < 5) {
    console.log("Iteration " + i);
    i++;
}

do...while Loop

let i = 0;
do {
    console.log("Iteration " + i);
    i++;
} while (i < 5);
