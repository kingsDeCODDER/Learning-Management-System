Java Arrays

What are Arrays?

In Java, arrays are used to store multiple values in a single variable. Instead of declaring separate variables for each value, you can declare an array that holds a collection of values of the same type.

Arrays have a fixed size once they are created, meaning their length cannot be changed.

Declaring and Initializing Arrays

Arrays are declared with square brackets []. You can declare and initialize an array in two steps:

// Declare an array of integers
int[] numbers;

// Initialize the array with values
numbers = new int[]{10, 20, 30, 40, 50};

Alternatively, you can declare and initialize the array in one line:

int[] numbers = {10, 20, 30, 40, 50};

Accessing Array Elements

Each element in an array can be accessed using its index, which starts at 0. For example:

int[] numbers = {10, 20, 30, 40, 50};
System.out.println(numbers[0]);  // Outputs 10
System.out.println(numbers[3]);  // Outputs 40

If you try to access an index that does not exist, Java will throw an ArrayIndexOutOfBoundsException.

Looping Through Arrays

You can loop through arrays using a for loop or an enhanced for loop (also known as a "for-each" loop).

Using a For Loop

int[] numbers = {10, 20, 30, 40, 50};

for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}

This loop prints all the elements of the array.

Using a For-Each Loop

int[] numbers = {10, 20, 30, 40, 50};

for (int number : numbers) {
    System.out.println(number);
}

The for-each loop is a more concise way to iterate over all the elements in an array.

Multidimensional Arrays

Java supports multidimensional arrays, such as 2D arrays (arrays of arrays). These are often used to represent matrices or tables:

int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

You can access elements of a 2D array using two indices:

System.out.println(matrix[1][2]);  // Outputs 6

Array Length

To get the length (number of elements) of an array, you can use the .length property:

int[] numbers = {10, 20, 30, 40, 50};
System.out.println("Length of the array: " + numbers.length);  // Outputs 5

Common Array Operations

Here are some common operations you can perform with arrays:

Finding the Maximum or Minimum Value

int[] numbers = {10, 20, 30, 40, 50};
int max = numbers[0];
int min = numbers[0];

for (int number : numbers) {
    if (number > max) {
        max = number;
    }
    if (number < min) {
        min = number;
    }
}
System.out.println("Maximum: " + max);
System.out.println("Minimum: " + min);

Copying an Array

You can copy an array using System.arraycopy():

int[] numbers = {10, 20, 30};
int[] copy = new int[3];
System.arraycopy(numbers, 0, copy, 0, 3);

for (int num : copy) {
    System.out.println(num);
}
