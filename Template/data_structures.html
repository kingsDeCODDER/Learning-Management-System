Python Data Structures

1. Lists

A list in Python is an ordered collection of items that is mutable, meaning you can change, add, or remove items after the list has been created. Lists allow duplicate elements.

Example:

# Creating a list
my_list = [1, 2, 3, 4, 5]

# Accessing elements
print(my_list[0])  # Output: 1

# Modifying an element
my_list[2] = 10
print(my_list)  # Output: [1, 2, 10, 4, 5]

# Adding a new element
my_list.append(6)
print(my_list)  # Output: [1, 2, 10, 4, 5, 6]

2. Tuples

A tuple is similar to a list, but tuples are immutable. Once you create a tuple, you cannot change, add, or remove items from it. Tuples are used when the data should not be modified.

Example:

# Creating a tuple
my_tuple = (1, 2, 3, 4, 5)

# Accessing elements
print(my_tuple[1])  # Output: 2

# Trying to modify an element (will raise an error)
# my_tuple[1] = 10  # This will raise a TypeError

# Tuples can be used for returning multiple values from a function
def get_coordinates():
    return (10, 20)

x, y = get_coordinates()
print(x, y)  # Output: 10 20

3. Sets

A set is an unordered collection of unique items. Unlike lists and tuples, sets do not allow duplicate elements, and the order of elements is not guaranteed.

Example:

# Creating a set
my_set = {1, 2, 3, 4, 5}

# Adding elements to a set
my_set.add(6)
print(my_set)  # Output: {1, 2, 3, 4, 5, 6}

# Duplicate elements are automatically removed
my_set.add(3)
print(my_set)  # Output: {1, 2, 3, 4, 5, 6}

# Removing an element
my_set.remove(4)
print(my_set)  # Output: {1, 2, 3, 5, 6}

4. Dictionaries

A dictionary is an unordered collection of key-value pairs. Each key is unique, and you use the key to access the corresponding value. Dictionaries are mutable and very useful for storing related data.

Example:

# Creating a dictionary
my_dict = {'name': 'John', 'age': 25, 'city': 'New York'}

# Accessing a value
print(my_dict['name'])  # Output: John

# Modifying a value
my_dict['age'] = 26
print(my_dict)  # Output: {'name': 'John', 'age': 26, 'city': 'New York'}

# Adding a new key-value pair
my_dict['email'] = 'john@example.com'
print(my_dict)  # Output: {'name': 'John', 'age': 26, 'city': 'New York', 'email': 'john@example.com'}

5. Conclusion

Python provides several built-in data structures, each designed for different use cases. Lists and dictionaries are commonly used for flexible and mutable collections, while tuples are used when immutability is important. Sets are helpful when dealing with unique elements.
